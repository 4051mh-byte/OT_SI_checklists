4. SLP·PT와의 교차점

🗣 SLP에게 — 기초인지와 언어의 교차점

인지 영역SLP 관련 포인트
위치·방향 개념
(위/아래/안/밖/앞/뒤)		• 공간 전치사 = 언어 이해의 핵심
• OT에서 신체 기반 경험 → SLP에서 언어적 표현 연결
• "위에 올려놔" 수행 불가 시 → 지시 이해 문제 vs 개념 부재 감별 필요
분류·범주화
(색·형태·기능·상위범주)		• 범주 이름대기 = 어휘력 핵심
• REVT 결과와 OT 분류 과제 교차 확인
• "먹는 것끼리" 불가 → 범주 어휘 부재 vs 분류 추론 부재 감별
수 개념·수리언어
(기수성, 서수, 비교)		• "더 많은/적은/같은" = 수학적 어휘
• 서수("첫 번째") = 언어 발달 직결
• 수 개념 어휘 15~30개는 SLP 어휘 중재에 포함 권고
순서화·인과• 사건 순서 = 이야기 이해·구성력 직결
• 그림카드 순서 배열 실패 → story retell 확인
• "왜냐하면" 사용 = 인과 접속사 발달 지표
같다/다르다·비교• 개념적 비교("어떻게 같아?") = 추론적 언어 기초
• 비교·대조 표현 = 학령기 학업 언어 핵심
• 시각적 비교(OT) OK + 언어적 비교(SLP) 실패 → 주의
작업기억·과제이해• 숫자폭 = 문장 이해 길이 예측 지표
• 복합 지시 이해 불가 → 기억용량 vs 언어이해 감별
• 과제 지시 요약 불가 → 이해력 vs 표현력 감별
🔑 SLP ↔ OT 협력 핵심 질문:
"이 아동이 개념을 모르는 건가요(OT), 표현을 못하는 건가요(SLP)?"
→ 비언어 과제(OT)와 언어 과제(SLP) 결과를 비교하면 감별 가능

🏃 PT에게 — 기초인지와 운동의 교차점

인지 영역담당PT 관련 포인트
거리·깊이 지각OTPT• 깊이 지각 = 계단, 점프, 장애물 회피 직결
• 거리 추정 = 공 던지기/잡기의 기초
• PT에서 운동 과제 시 지각 실패 vs 운동 실패 감별 필요
• OT에서 정적 지각 평가 → PT에서 동적 수행 확인
위치·방향 개념OTPT• "앞으로/뒤로/옆으로" = 운동 지시 기본
• 공간 개념 부재 → PT 지시 수행 자체 곤란
• 신체 기반 공간 경험은 PT에서 가장 자연스럽게 제공 가능
• OT에서 개념 평가 → PT에서 신체 경험으로 강화
대칭·좌우 인식OTPT• 양측 대칭 운동 = 신체 대칭 인식 기초
• 한쪽 동작 → 반대쪽 모방 = 신체 대칭 전이
• 좌우 인식 불가 → PT 양측 협응 활동에서 확인
• OT 시지각 대칭 과제 결과와 PT 양측 협응 교차 확인
순서화·절차기억OTPT• 운동 순서(달리기→점프→착지) = motor planning
• 순서화 실패 → 복합 운동 단계 건너뜀/혼동
• 시각 카드로 운동 순서 제시 → OT 순서화 목표 연계
• OT에서 인지적 순서화 → PT에서 운동 순서로 전이
계열화·힘 조절OTPT• 힘 단계화(살살→세게) = 계열화의 운동 적용
• 거리 단계화(가까이→멀리) = 거리지각+계열화 통합
• "조금 더/조금 덜" 조절 = 계열화 연계
• OT 크기/길이 계열화 과제와 PT 힘/거리 조절 연계
억제조절·규칙유지OTPT• "멈춰" 신호 반응 = 안전 직결
• 게임 규칙 유지 = PT 그룹활동 참여 기초
• 충동억제 부족 → PT 활동 중 안전사고 위험
• OT에서 정적 억제 과제 → PT에서 동적 억제(운동 중 멈춤) 확인
🔑 PT ↔ OT 협력 핵심 질문:
"이 아동이 지각을 못하는 건가요(OT), 운동 실행을 못하는 건가요(PT)?"
→ 정적 판단 과제(OT)와 동적 수행 과제(PT)를 비교하면 감별 가능

5. 감각처리(SI)와의 교차점

🖐 감각처리(SI)에게 — 기초인지와 감각처리의 교차점

인지 영역담당SI 관련 포인트
주의력·집중
(선택적·지속적 주의)		OTSI• 전정감각 저등록 → 각성 수준 저하 → 지속적 주의 실패로 오인 가능
• 촉각/청각 과민 → 감각 방어 반응에 에너지 소진 → 선택적 주의 저하
• 고유수용감각 저등록 → 자세 불안정 → 착석 유지 불가 → 주의력 문제로 오인
• 감각조절 실패 시 인지 과제 수행 자체가 불가 → ADHD 선별 전 SI 평가 선행 권고
지각·변별
(형태, 크기, 깊이)		OTSI• 촉각 변별 저하 → 도구 조작 과제(퍼즐, 블록) 수행 실패 → 시지각 문제로 오인 가능
• 전정-시각 통합 부족 → 시각 추적 불안정 → 전경-배경 분리, 형태항상성 평가에 영향
• 고유수용감각 피드백 부족 → 그리기/쓰기 시 힘 조절 실패 → 시각-운동 통합 저하와 감별 필요
기억력
(작업기억, 순서기억)		OTSI• 감각 과부하 상태 → 작업기억 용량 축소 (감각 처리에 인지 자원 소모)
• 촉각·고유수용감각 기반 신체 기억 = 절차기억의 토대
• 감각 조절 안정 후 숫자폭 재측정 시 향상되면 → 기억력 자체 문제 아님, SI 중재 우선
실행기능
(억제, 계획, 유연성)		OTSI• 감각추구 행동(만지기, 움직이기) → 충동억제 실패로 오인 가능
• 전정감각 과민 → 새로운 환경/자세 회피 → 인지적 유연성 부족으로 오인
• 자기조절 실패의 근원이 감각조절인지 인지적 억제인지 감별 필수
• 감각 다이어트 적용 후 억제 과제 재평가 → 변화 있으면 SI 중재 우선
사고·추론
(분류, 문제해결)		OTSI• 촉각 방어 → 다양한 재질 탐색 회피 → 분류 경험 자체 부족
• 고유수용감각 통합 부족 → 도구 사용 시행착오 제한 → 문제해결 기회 감소
• 감각 경험 기반 학습(만지고, 느끼고, 움직이며 배우기)이 제한되면 추상적 개념 형성 지연
도구 사용·과제 수행
(가위, 필기구, 퍼즐)		OTSI• 촉각 방어 → 특정 재질 회피 → 점토, 풀, 가위 과제 거부를 인지 문제로 오인
• 양측 협응 부족(SI 기반) → 가위질, 종이접기 실패 → 시지각·계획 문제와 감별 필요
• "이 아동이 하기 싫은 건가요, 못하는 건가요, 감각적으로 불편한 건가요" 감별
🔑 SI ↔ OT(인지) 협력 핵심 질문:
"이 아동이 인지적으로 못하는 건가요, 감각 입력 자체가 방해하는 건가요?"
→ 감각 환경 조절 전/후 동일 과제 수행 비교로 감별 가능
→ 감각 다이어트 적용 후 인지 과제 수행이 향상되면 SI 중재 우선

7. 보호자 예상 질문 & 권장 답변

❓ 보호자 예상 질문 — "이렇게 물어보시면 이렇게 답해주세요"

#예상 질문권장 답변치료사 참고 (보호자에게는 말하지 않음)
1"우리 아이가 장애인 건가요?""지금 말씀드린 건 '진단'이 아니라 '관찰'이에요. ○○이가 이 부분에서 또래보다 연습이 좀 더 필요하다는 뜻이에요. 더 정확하게 알기 위해 정밀 평가를 함께 진행해 보면, 어떤 도움이 가장 효과적인지 알 수 있습니다."• Red Flag ≠ 진단임을 반복 강조
• "선별 → 평가 → 판단"의 단계가 있음을 설명
• 보호자가 '장애' 단어에 강한 정서 반응 보이면 즉시 공감 먼저, 정보 전달은 다음 면담으로
2"크면 자연스럽게 괜찮아지지 않나요?""어떤 부분은 자연스럽게 발달하기도 해요. 다만 지금 관찰된 부분은 기다리기보다 연습해주는 게 훨씬 효과적인 영역이에요. 특히 이 시기에 도와주면 나중에 학교에서 훨씬 수월해집니다."• "기다려 보자"는 접근이 가장 위험한 Red Flag: 공동주의 부재, 숫자폭 2 미만, 기억전략 자발사용 없음
• 조기 중재 효과 관련 근거 데이터를 준비해두면 설득력 높음
• 연령이 높을수록(8세+) 자연 해소 가능성 ↓ 임을 인지
3"제가 뭘 잘못한 건가요?"
"맞벌이라서 그런 건가요?"		"절대 부모님 탓이 아닙니다. 아이마다 발달 속도와 강점이 다르고, 이건 양육 방식과는 별개의 문제예요. 오히려 이렇게 일찍 관심을 가져주신 것 자체가 ○○이에게 큰 도움이 됩니다."• 죄책감 표현은 보호자가 충분히 관심을 갖고 있다는 신호 → 강점으로 리프레이밍
• 양육 환경 탓으로 돌리는 언급 절대 금지
• 보호자 자신의 정서적 소진에도 관심 기울이기
4"스마트폰/영상을 많이 봐서 그런 건가요?""미디어 노출이 주의력이나 언어 발달에 영향을 줄 수 있다는 연구는 있어요. 다만 지금 관찰된 부분의 원인을 한 가지로 단정하기는 어렵습니다. 미디어 시간을 조절하시면서, 동시에 저희와 함께 연습하면 가장 좋은 결과를 기대할 수 있어요."• 미디어를 유일한 원인처럼 말하면 보호자 방어 반응 ↑
• "미디어 줄이면 해결된다"는 오해 방지 필요
• 미디어 대신 상호작용 놀이 시간 늘리기를 구체적으로 안내
• AAP 권고(2세 미만 미디어 제한, 2~5세 1시간/일) 참고 가능
5"치료를 얼마나 오래 받아야 하나요?""아이마다 속도가 달라서 지금 정확한 기간을 말씀드리기는 어렵지만, 보통 3개월 단위로 목표를 세우고 얼마나 성장했는지 함께 확인해요. 변화가 보이면 그때마다 계획을 조정합니다."• 기간을 특정하면 "그때까지 안 되면?" 불안 유발
• 3개월 단기 목표 → 재평가 → 조정 사이클로 설명
• Stage 전환 기준을 보호자 언어로 풀어서 공유하면 진전도 이해에 도움
6"다른 병원에서는 괜찮다고 했는데요?""그 평가 결과도 중요한 정보예요. 다만 평가 시점, 환경, 도구에 따라 결과가 다를 수 있어요. 저희가 관찰한 부분을 함께 공유드리면, 종합적으로 판단하는 데 도움이 될 겁니다."• 이전 평가를 부정하거나 깎아내리지 않기
• "관점의 차이"로 프레이밍 — 발달 선별 vs 정밀 인지 관찰은 초점이 다름
• 가능하면 이전 평가 보고서를 공유받아 비교 분석
7"집에서 제가 뭘 해주면 되나요?""좋은 질문이에요! 거창한 것보다 일상 속 작은 연습이 가장 효과적이에요. 예를 들어 (연령별 예시 1~2가지 제공). 다음 시간에 가정용 활동지를 드릴게요."• 연령별 가정 활동 예시:
  · 2~3세: 사물 이름대기 놀이, "○○ 위에 올려봐" 지시 놀이
  · 4~5세: 분류 놀이(빨래 색깔별 나누기), 순서 말하기
  · 6~7세: 보드게임(규칙 유지), 요리 시 계량(수 개념)
  · 8세+: 일기 쓰기(순서화), 할 일 목록 만들기(계획)
• 과제를 너무 많이 주면 부담 → 1~2가지만 안내
8"학교에 알려야 하나요?"
"특수교육을 받아야 하나요?"		"지금 바로 특수교육이 필요한 건 아닐 수 있어요. 다만 담임 선생님과 정보를 공유하시면 교실에서도 ○○이를 더 잘 도와줄 수 있어요. 어떻게 전달하면 좋을지 저희가 함께 정리해 드릴게요."• 보호자가 '특수교육' 단어에 거부감 강하면 → "학습 도움"으로 순화
• 담임 소통 시 치료사 소견서/관찰 요약본 제공 제안
• Stage 3 이후 학업 영향 뚜렷한 경우 → 특수교육 지원 연계 필요성을 서서히 안내
• IEP/통합교육 관련 학교 절차 기본 정보 준비
9"ADHD 약을 먹어야 하나요?""약 처방은 소아과/소아정신과 전문의 선생님이 판단하시는 부분이에요. 저희는 먼저 행동적·인지적 접근으로 도와드리고, 필요하시면 전문의 상담을 연결해 드릴 수 있어요."• 치료사가 약물에 대해 의견을 내지 않기 — 전문의 영역 존중
• 억제조절 80%+ 실패 / 충동응답 70%+ 시 → 선별 검사 후 소아과 의뢰 경로 안내
• "약 vs 치료" 이분법 지양 → 병행 가능함을 설명
10"또래보다 얼마나 느린 건가요?""'빠르다/느리다'보다는 '이 부분에서 더 도움이 필요하다'고 이해해 주시면 좋겠어요. ○○이만의 속도가 있고, 지금 집중해서 연습하면 격차가 줄어드는 부분이에요."• 또래 비교 수치(백분위, 개월 수 차이)를 직접 말하면 보호자 충격 큼
• "6개월 지연" 같은 표현 대신 → 현재 수행 수준과 다음 목표로 설명
• 정밀 평가 결과가 있을 때만 구체적 수치를 공유하되, 반드시 해석과 함께 전달
11"아이 앞에서 이런 얘기 해도 되나요?""좋은 질문이에요. 어린 아이는 대화 내용을 다 이해하지 못하지만, 학령기 이상은 분위기와 감정을 느낄 수 있어요. 구체적인 내용은 아이 없이 따로 말씀드리고, 아이에게는 '더 잘할 수 있도록 연습하는 거야' 정도로 알려주시면 됩니다."• 4세 이상부터는 아동 동석 여부를 사전에 보호자와 협의
• 8세 이상은 아동 본인에게도 연령에 맞는 설명 필요 (자기이해 촉진)
• 아동 앞에서 부정적 라벨 사용 시 학습된 무력감 위험 → 면담 시 항상 유의
12"저도 어릴 때 그랬는데 괜찮았어요.""그 경험을 말씀해 주셔서 감사해요. 실제로 가족력이 있을 수 있고, 자연스럽게 나아지는 경우도 있어요. 다만 지금은 예전보다 학교에서 요구하는 수준도 달라졌고, 미리 도와드리면 ○○이가 덜 힘들 수 있어요."• 보호자 본인의 경험을 부정하지 않되, 현재 교육 환경의 차이를 설명
• 가족력 정보는 평가에 중요한 단서 → 추가 질문으로 연결
• "괜찮았다"가 실제로는 보상 전략을 스스로 만든 경우도 → 인정 후 아동에게는 더 빨리 도와줄 수 있음을 강조

8. 즉시 팀 공유 & 평가도구 참조

📢 즉시 팀 공유가 필요한 6가지 상황

1. 안전 관련
깊이/거리 지각 부재로
낙상·충돌 위험 있을 때
→ PT·보호자에게 즉시 공유
2. 지시 이해 불가
공간 전치사·과제이해 불가로
모든 치료사 지시 수행 곤란
→ OT·SLP 협력 평가 우선
3. 전반적 인지 지연
3개 영역+ 동시 Red Flag 시
종합 발달 평가 의뢰
→ 전 팀원 회의 소집
4. Stage 전환 판단
전환 기준 미달 영역 발견 시
팀 회의 → 유지/전환 결정
→ 전 팀원 Stage 공유
5. 영역 간 불일치
시각적 과제 OK + 언어적 실패
(또는 그 반대)
→ OT·SLP 감별 평가 필요
6. 충동/억제 심각
억제조절 극심 → ADHD 선별
+ 안전사고 위험
→ 전 팀+소아과 의뢰
